# Product Dev Workflow

This workflow bundle composes the first product-development path:

1. Shape the request with `superpowers:brainstorming`.
2. Review the requirement direction with `ponyrace`.
3. Write the implementation plan with `superpowers:writing-plans`.
4. Preserve file-change evidence with `pony-trail`.

It installs the workflow contract first. Automatic step execution is intentionally deferred.
